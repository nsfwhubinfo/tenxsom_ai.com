## Pro Se Filing Steps

### Step 1: Provisional Filing ($75)
1. Log into Patent Center: https://patentcenter.uspto.gov/
2. Select "New application" → "Provisional"
3. Upload specification as DOCX
4. Complete applicant information
5. Certify micro-entity status (Form SB/15B)
6. Pay $75 fee
7. Receive Application Number

### Step 2: Prior Art Search (Months 1-6)
- [ ] Search USPTO database
- [ ] Search Google Patents
- [ ] Search IEEE/ACM databases
- [ ] Document all findings
- [ ] Prepare IDS form

### Step 3: Claim Refinement (Months 6-9)
- [ ] Analyze prior art
- [ ] Refine claims for novelty
- [ ] Consider professional review ($500-1000)
- [ ] Prepare formal drawings if needed

### Step 4: Non-Provisional Filing (Month 11)
1. Log into Patent Center
2. Select "New application" → "Nonprovisional"
3. Claim priority to provisional
4. Upload:
   - Specification.docx
   - Claims.docx
   - Abstract.docx
   - Drawings.pdf (if any)
   - ADS (Application Data Sheet)
5. Certify micro-entity status
6. Pay fees ($500 total)

## Red Flags to Avoid

1. ❌ Don't claim abstract ideas without technical implementation
2. ❌ Don't use functional claiming without structure
3. ❌ Don't forget to disclose best mode
4. ❌ Don't add new matter after filing
5. ❌ Don't miss 12-month deadline
