How to Fix Navigation in GoDaddy Website Builder

Problem: GoDaddy Website Builder uses JavaScript to load pages within the current page instead of normal navigation. This creates a poor user experience and can break browser history.

Solution Options:

Option 1: Add Custom JavaScript (Recommended)

  1. Log into your GoDaddy Website Builder
  2. Go to Settings → Advanced → Custom Code
  3. Add this code to the Footer Code section:
<script>
// Navigation fix for normal page transitions
document.addEventListener('DOMContentLoaded', function() {
    setTimeout(function() {
        const navLinks = document.querySelectorAll('a[href]:not([href^="#"])');
        navLinks.forEach(link => {
            const newLink = link.cloneNode(true);
            link.parentNode.replaceChild(newLink, link);
            newLink.addEventListener('click', function(e) {
                e.stopPropagation();
                const href = this.getAttribute('href');
                if (href && href !== '#') {
                    e.preventDefault();
                    window.location.href = href;
                }
            }, true);
        });
    }, 1000);
});
</script>

Option 2: Edit Navigation Links Directly

  1. In the Website Builder, click on each navigation link
  2. Change the link type from "Section" to "URL"
  3. Enter the full URL (e.g., https://devprompt.solutions/about)
  4. Make sure "Open in new window" is unchecked

Option 3: Use HTML Blocks

  1. Remove the default navigation widget
  2. Add an HTML block in its place
  3. Create custom navigation HTML:
<nav style="display: flex; gap: 20px; padding: 20px;">
    <a href="/" style="color: white; text-decoration: none;">Home</a>
    <a href="/about" style="color: white; text-decoration: none;">About</a>
    <a href="/services" style="color: white; text-decoration: none;">Services</a>
    <a href="/contact" style="color: white; text-decoration: none;">Contact</a>
</nav>

Testing Your Fix

  • Click on navigation links - they should load new pages normally
  • Use browser back button - it should work properly
  • Check that URLs in the address bar change when navigating
Note: GoDaddy may update their platform which could affect these fixes. If the issue persists, consider migrating to a different hosting platform that gives you more control over your site's behavior.
